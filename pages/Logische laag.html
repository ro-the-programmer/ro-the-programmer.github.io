OS

Het Operating System(OS) is een systeem die taken uitvoert zodat de computer toegangbaar is voor de gebruiker. Zo heb je bijvoorbeeld MacOS van Apple, die je toegang geeft tot je files en je apps laat openen door een uniek gebruikersyteem. De OS is dus eigenlijk een middleman die communiceert tussen de gebruiker en de software en hardware.

DBMS

Een DataBaseManagementSystem is een type software dat data opslaat en organiseert/analyseert. Deze data wordt opgeslagen in een database. De DBMS is de beheerder van deze data.
