Apps

De app(afkorting van Application-software) is een programma dat niet met de werking van de computer te maken heeft. Een voorbeeld van een app is bijvoorbeeld een mp4 speler of google chrome. Binnen de apps heb je twee groepen: de open source apps en de closed source apps. Bij open source apps is de code door iedereen te bekijken, en zijn mensen ook vrij om hun eigen versies of verbeteringen te maken. Bij closed source apps is de code juist niet beschikbaar tot het publiek. Dit wordt gedaan door bedrijven als ze code hebben die zij alleen willen gebruiken. Meeste grote games zoals Mario zijn closed source, omdat het moeder-bedrijf(Nintendo) wilt alleen zij controle hebben over de game.

API

Application Programming Interface, verkort: API, is een connectie die computer software of computers verbindt. Als je bijvoorbeeld twee apps hebt, en je wilt dat ze kunnen communiceren, heb je een API nodig. De API is altijd buiten zicht van de computer-gebruiker, en werkt dus achter de schermen om de ervaring te verbeteren. Een API komt vaak in de vorm van een bibliotheek van definities die verschillende programma's kunnen gebruike om te communiceren zonder dat ze hoefen te weten hoe het andere programma werkt.
