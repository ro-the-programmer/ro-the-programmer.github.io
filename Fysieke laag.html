CPU

De CPU, ookwel de processor of Central Processing Unit genoemed, is een van de belangrijkste componenten van een computer. De CPU bestaat uit meerdere dingen, de Arithmatic Logic Unit, de Floating Point Unit, de Central Unit en processor registers.

  • De ALU doet wiskundige taken en simpele logische functies, zoals de AND functie. De AND functie kijkt naar twee bytes, en als ze bijde aan staan, de gaat de AND ook aan.
  • De FPU doet wiskundige take op komma getallen in plaats van hele getallen zoals bij de ALU.
  • De CU is de zogenaamde leider van de CPU, het zegt namelijk tegen de andere componenten wat en wanneer ze iets moeten doen.
  • De processor registers zijn kleine, snel toegankelijke plaatsen van geheugen, deze worden gebruikt wanneer een stukje data veel bekeken moet worden, zo hoeft de CPU niet de main memory te vragen naar een stuk data dat lang duurt te vinden.

De CPU is gebouwd op een IC(integrated circuit), ookwel een computer chip genoemd. Deze IC is gemaakt van silicoon, waarin je transistoren en capacitors en resistors in kan schrijven met behulp van een gespecialiseerde laser.

Kloksnelheid

De kloksnelheid is de snelheid waarmee bepaald wordt hoe vaak per seconde een actie uitgevoerd wordt. De snelheid is consistent en wordt aangegeven door door een klein componentje: de kristaloscilator. De kristaloscilator heeft een klein stukje kwartz erin dat een frequentie geeft onder elektriciteit. Ditzelfde mechanisme geven elektrische kwartz horloges ook hun frequentie waarmee zij de tijd tellen. De moderne processor heeft een kloksnelheid van 3ghz, dat is dus 3 miljard acties per seconde. Elke computer heeft een ingestelde kloksnelheid, deze kan worden aangepast om de computer te overklokken of te onderklokken. Bij overklokken gaat de kloksnelheid sneller dan wat bedoeld is, waardoor de componenten meer energie in nemen en heter worden, waardoor er schade kan komen op de computer.

IPS

IPS, ookwel Instructies Per Seconde, is hoe snel een computer een instructie uit kan voeren. De IPS is niet te verwarren met de kloksnelheid, want een instructie kan meerdere acties nodig hebben. De IPS wordt vaak gemeten in MIPS(Miljoenen Instructies Per Seconde). De IPS is vaak ook niet de beste manier om computersnelheid te meten, want de computerarchitectuur heeft ook veel invloed op de snelheid van een computer.

RAM

De RAM, ookwel Random Access Memory, is het tijdelijke geheugen dat recht naast de CPU ligt. De RAM kan worden geaccessed op een random point, dit betekent dat je snel data eruit kunt halen, en dat de positie van die data niet uitmaakt. Dit is zeer handig voor CPU snelheid, zo hoeft de data niet ver te reizen.

ROM

De ROM ookwel de Read Only Memory wordt gebruikt door de CPU om veel gebruikte data te lezen. De data in de ROM kan dus niet gewist worden. Deze niet-wisbare data wordt ookwel firmware genoemt. Wanneer je het over een ROM tegen mij hebt, dan denk ik meteen aan de ROM disks van de vroege jaren 2000. Toen werden video-games verandered door induviduele personen om hun eigen versie te maken. Deze versie gingen dan uploaden op een ROM disk en gingen ze aan hun vrienden uitlenen om te kunnen spelen. Natuurlijk werdt dit niet gerwaardeert door de originele fabrikanten van de video-games, maar de ROM disks legde wel de basis voor de huidige video modificatie.
